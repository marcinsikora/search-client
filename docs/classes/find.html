Class Find

The Find service queries the search-engine for search-matches for the given query.

It is normally used indirectly via the SearchClient class.

fetch

  • fetch(query?: Query, suppressCallbacks?: boolean): Promise<IMatches>
    • Defined in Find/Find.ts:51

    Fetches the search-result matches from the server. Note that if a request callback has been setup then if it returns false the request is skipped.

    Parameters

    • Default value query: Query = new Query()

      The query-object that controls which results that are to be returned.

    • Default value suppressCallbacks: boolean = false

      Set to true if you have defined callbacks, but somehow don't want them to be called.

    Returns Promise<IMatches>

    a Promise that when resolved returns a string array of suggestions (or undefined if a callback stops the request).
